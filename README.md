📚 Краткое описание программы
Программа "Weather Diary" представляет собой приложение для ведения дневника погоды с использованием графического интерфейса на базе библиотеки **Tkinter**. Пользователи могут добавлять записи о погоде, указывая дату, температуру, описание и информацию о наличии осадков. Также предусмотрены функции фильтрации записей по дате и температуре. Данные сохраняются в формате JSON в файле `weather_diary.json`.
🚀 Как запустить приложение
1. Убедитесь, что у вас установлен Python и библиотека Tkinter. - Tkinter обычно включен в стандартную библиотеку Python. Если у вас его нет, можно установить, используя пакетный менеджер вашей ОС.
Скопируйте приведённый выше код в файл с расширением .py, например, weather_diary.py.

Запустите скрипт через терминал или командную строку:

python weather_diary.py
Откроется окно приложения, где вы сможете вносить данные о погоде и фильтровать записи.

💡 Примеры использования (возможные тесты)
- **Добавление записи**: - Введите дату: `2023-10-01` - Температура: `15` - Описание: `Солнечно` - Установите флажок "Осадки" (если актуально). - Нажмите кнопку "Добавить запись".
Фильтрация по дате:

Введите в поле фильтрации дату: 2023-10-01
Нажмите кнопку "Фильтровать по дате". В таблице отобразится запись только за указанную дату.
Фильтрация по температуре:

Введите в поле фильтрации температуру: 10
Нажмите кнопку "Фильтровать по температуре". В таблице отобразятся записи с температурой выше 10°C.
